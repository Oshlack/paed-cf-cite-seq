This site presents the code and results of the analyses described in the pre-print: “Multimodal single-cell analysis of bronchoalveolar lavage from children with cystic fibrosis reveals novel cell phenotypes”.

All the code and results of this analysis are available from GitHub at https://github.com/Oshlack/paed-cf-cite-seq. To reproduce the complete analysis follow the instructions on the getting started page. The raw single cell RNA-seq and CITE-seq count data generated for this study can be downloaded as RDS files from .
